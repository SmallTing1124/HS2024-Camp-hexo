    測試 Hexo Markdown
    Markdown測試

    Markdown是一種輕量級的標記語言，由約翰·格魯伯（John Gruber）於2004年創立，旨在讓書寫格式化文本變得簡單且可讀性高。Markdown使用直觀的符號來標示不同的文本格式，如粗體、斜體、OpenAI 、> 引用、代碼區塊和列表等，使得作者能專注於內容創作而非格式設置。這種語法特別適合撰寫博客文章、技術文檔、電子郵件和網頁內容。

    Markdown的優勢在於其簡單易學且易於轉換為多種格式，如HTML、PDF和Word文檔。許多流行的版本控制平台如GitHub、GitLab都支持Markdown，讓協作和文檔編寫變得更加便利。此外，眾多的編輯器和應用程式也支援Markdown，即時預覽功能使得撰寫和編輯更加直觀。總體而言，Markdown憑藉其簡單、靈活和強大的功能，成為內容創作者和開發者們的首選工具。
